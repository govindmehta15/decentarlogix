# DecentraLogix

A Web3-powered logistics platform built on Ethereum blockchain.

## Project Structure

```
DecentraLogix/
├── smart-contracts/     # Solidity contracts + Hardhat
├── backend/             # Node.js API layer
├── frontend/            # React Web3 UI
└── docs/                # Documentation
```

## Architecture Overview

### Smart Contracts Layer
- **Technology**: Solidity + Hardhat
- **Purpose**: Core business logic on blockchain
- **Key Features**: Shipment tracking, payment escrow, stakeholder management

### Backend API Layer
- **Technology**: Node.js (Express/Fastify)
- **Purpose**: Bridge between blockchain and frontend, IPFS integration
- **Key Features**: Event indexing, data aggregation, IPFS file management

### Frontend Web3 UI Layer
- **Technology**: React + Ethers.js
- **Purpose**: User interface for interacting with the platform
- **Key Features**: Wallet connection, shipment management, real-time tracking
